# Restaurant Web App

A comprehensive, multi-page frontend website for a restaurant. It features a modern design and interactive elements to showcase the menu, locations, offers, and a functional shopping cart.

## Features
- **Extensive Pages**: Includes Home, Menu, Cart, Offers, Locations, Gallery, and Testimonials.
- **Shopping Cart**: Interactive cart functionality for ordering food (`cart.html` & `script.js`).
- **Responsive Design**: Styled with custom CSS for all screen sizes.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started
Simply clone the repository and open `index.html` in your favorite web browser to view the site. No build process required!
